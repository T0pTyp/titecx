# Ti-Tecx Website

Moderne, statische One-Page-Website für Ti-Tecx.

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Build & Export

```bash
npm run build
```

Der statische Export befindet sich im `out/` Verzeichnis und kann auf jedem statischen Hosting-Service deployt werden.
