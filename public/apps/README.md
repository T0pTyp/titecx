# App Icons

Platziere hier die PNG-Icons für die App-Karten im Karussell.

## Benötigte Dateien:

- `web-app.png` - Icon für "Web & App"
- `daten.png` - Icon für "Daten & Auto"
- `ki.png` - Icon für "KI & Analyse"
- `architektur.png` - Icon für "Architektur"
- `devops.png` - Icon für "DevOps"

## Empfohlene Spezifikationen:

- **Format**: PNG mit Transparenz
- **Größe**: 192x192px oder 256x256px (quadratisch)
- **Stil**: Modern, minimalistisch, passend zum Dark-Mode-Design
- **Farben**: Blau-Töne passend zum Theme (z.B. #3B82F6)

Die Icons werden automatisch auf 80-96px skaliert und mit 3D-Effekten angezeigt.
